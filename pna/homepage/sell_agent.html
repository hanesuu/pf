Pay&Api대리점으로 대박내는 법

개발자 회원들에게 페이앤 리셀러를 적극 홍보하세요.
짱짱한 연금성 수익이 기다립니다~

  • STEP 1
    대리점
    무료 회원가입
    회원가입 후 대리점 전용 관리자 페이지에 접속이 가능합니다.
    가입 후 귀사 전용 리셀러 가입링크가 생성되며, 이 링크로 가입한 회원은
    귀사의 리셀러로 자동 등록됩니다.
  • STEP 2
    이용 가능 API 및
    공급원가 확인하기
    로그인 후 API별 내 원가 조회가 가능합니다.
  • STEP 3
    리셀러 등록
    귀사 전용 가입링크를 배너로 게시하거나 공유하여
    개발자 회원들의 셀프 가입을 유도할 수 있으며,
    리셀러 계정을 직접 등록할 수도 있습니다.
  • STEP 4
    판매점 API
    연동하기
    대리점 하부의 리셀러들이 판매점 API를 연동하거나,
    대리점이 직접 판매점 등록 후 API 연동을 대행할 수 있습니다.
    판매점 연동토큰과 매뉴얼을 참고해 API를 연동하세요.

간편 설정 끝.

이제 기다리기만 하면 아래와 같은 과정으로 수익이 들어옵니다.

판매점

유디페이

대리점

1. 판매점에서 유디페이로 결제수수료, 이용요금 지불
연동 후 Traffic에 따라 판매점은 API 결제수수료 및 유료 API의 이용요금을 유디페이로 지불합니다.
선불충전 API는 당월분 사용액이 당월에 결제되며,
후불결제 API는 당월분 사용액이 다음달 20일까지 납부됩니다.
2. 판매점에 대해 설정한 대리점 마진금액이 다음달 말일까지 일괄 지급
결제수수료 및 선불충전 API 마진은 다음달에, 후불결제 API 마진은 다다음달에 지급됩니다.
수익정산을 처음 받으실 때 간단한 서류 몇가지를 제출하셔야 합니다~
