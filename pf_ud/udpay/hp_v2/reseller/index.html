이런 분들이 유디페이 리셀러로 딱!

결제와 유료 API 연동이 필요한 고객을 많이 만나시는
개발자, 에이전시, 프리랜서에 최적화되어 있습니다.

  • 외주 개발자/프리랜서 박OO
    사업자 등록증이 없는 개인도 부업으로 수익을 낼 수 있어 좋아요.
  • 쇼핑몰 제작 에이전시 김OO
    쇼핑몰 제작과 동시에 간편하게 연동하여 쏠쏠하게 돈벌 수 있어요.
  • 결제 관련 솔루션 제작업체 이OO
    별도로 PG 가입 없이 통합 연동으로 업무 시간이 절약돼요.
개발자 회원이 많은 카페, 포럼, 커뮤니티 운영업체, 플랫폼 서비스 업체분들은 대리점이 되어 보세요.
리셀러를 모으고 리셀러를 통한 API 연동 마진이 생기면 대리점 수익이 자동으로 늘어납니다.
