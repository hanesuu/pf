api 지원

무통장 입금시 입금 내역을 자동으로 확인 처리하는 서비스입니다.

  • 입금확인
    무통장 입금 시 입금확인 및
    주문상태를 자동으로 변경
  • 계좌등록
    다수의 계좌 등록 가능
  • 실시간처리
    24시간 언제나 실시간 처리
  • 무료제공
    별도의 이용료 없이 무료로 제공

  • 이용안내
    입금통지 서비스는 각 은행의 홈페이지 혹은 고객센터로 신청 가능합니다.
    문자 알림 옵션 설정 시 24시간으로 설정해야 24시간 입금문자가 전달됩니다.
    문자 알림 옵션 설정 시 '예금잔액'과 '출금 시 통보'는 해제하시고, '0원 이상 입금통보'로 설정하세요.
    입금통지 서비스는 은행에서 제공하는 유료 부가 서비스로 각 은행별 일일 SMS 통지 건수 제한이 있을 수 있습니다.
    자세한 내용은 거래은행의 홈페이지에서 확인하시기 바랍니다.
    (최대 건수를 초과한 주문/입금건에 대해서는 판매자가 직접 주문 확인을 진행하셔야 합니다.)
    고객이 무통장 입금 시 주문자명과 입금자명, 금액이 동일해야 처리됩니다.

이용안내

1

'뱅크알람 > 설정' 메뉴에서 계좌 등록
1) 사용하실 계좌 추가
2) '다음' 클릭

2

거래은행 입/출금 SMS 알림 서비스 신청 후 '다음' 클릭
  • - 알림번호는 로그인 후 확인 필요
  • - 인증번호를 요구하지 않는 은행은 피드백 URL 입력으로 이동

3

SMS 인증번호를 거래은행 사이트에 입력하여 등록 완료 후 '다음' 클릭
  • - 원활한 등록 및 확인을 위해 거래은행의 SMS 신청 페이지와 뱅크알람 등록 페이지를 동시에 띄워놓고 진행해 주세요.

4

피드백 URL 입력 후 '완료' 버튼 클릭

5

뱅크알람 등록 완료
