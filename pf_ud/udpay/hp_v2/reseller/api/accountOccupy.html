계좌 점유 인증 서비스

api 지원

은행명, 계좌번호, 예금주 입력 시 해당 계좌에 1원을 이체 후 인증번호 입력을 통해 계좌의 점유여부를
거래전 확인하여 입출금 사고를 방지하는 서비스입니다.

  • POINT

    고객 계좌의 자동이체 등록, 환불 등
    현금 거래 시 주요하게 필요한 서비스

  • POINT

    보험사, 카드사, 캐피탈사, 도시가스업체,
    케이블 방송사, 여행사, 온라인쇼핑몰 등
    다양한 업체에 유용

  • POINT

    정기적으로 고객 계좌의 자동이체와 집금,
    비용 지급이 이루어지는 기관에 필수

계좌 점유 인증 적용 예시

  • STEP1
    은행선택, 계좌번호, 예금주
    입력 후 [계좌인증] 클릭

    유효계좌 시
