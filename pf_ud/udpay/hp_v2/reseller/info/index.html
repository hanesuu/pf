이용안내

결제와 유료 API 이용 Traffic 급증이 기대되는 고객사를
유디페이 판매점으로 적극 유치하여 수익을 창출해 보세요.

가입 후 수익정산까지

리셀러의 판매점 등록 시 판매자는 유디페이에 로그인하여 매출관리 및 정산관리가 가능합니다.

이용 가이드

부메뉴

  • 회원가입
  • 기본 마진 설정
  • 판매점 등록
  • 결제 및 마진 설정
  • 정산 서류 제출
  • 정산 안내

1

우측 상단의 '회원가입'을 선택합니다.

2

회원정보 입력 완료 후 '회원가입' 버튼을 선택합니다.

1

‘설정 > 기본 마진 설정’ 메뉴를 선택합니다.

2

기본 결제 마진과 기본 API 마진을 선택하여 마진을 입력합니다.

1

‘판매점 관리 > 판매점 등록’ 메뉴를 선택합니다.

2

직접 등록 또는 회원가입 링크를 통한 등록을 선택하여 판매점 등록을 진행합니다.
직접 등록 : 판매점 정보를 입력 후 '저장' 버튼을 선택하여 판매점을 등록합니다.

3

회원가입 링크를 통한 등록 : 판매점 담당자에게 회원가입 링크를 전달하여
담당자가 직접 판매점 정보를 입력하여 등록합니다.

1

‘판매점 관리 > 판매점 리스트’ 메뉴를 선택합니다.

2

판매점별 결제 설정 변경이 필요한 경우 ‘결제 설정’ 버튼을 선택하거나
‘상세 정보’ 페이지로 이동 후 ‘결제 설정’ 을 선택하여 결제 설정을 변경합니다.

3

판매점별 API 설정 변경이 필요한 경우 ‘API 설정’ 버튼을 선택하거나
‘상세 정보’ 페이지로 이동 후 ‘API 설정’ 을 선택하여 API 설정을 변경합니다.

1

‘설정 > 계정정보 > 서류제출’ 버튼을 선택합니다.

2

첨부서류를 업로드합니다.

제출하신 서류에 대한 내용을 확인한 후 보완이 필요한 경우 재요청 드리며, 검토기간은 영업일 기준 1~2일 소요됩니다.

3

검토완료 후 관리자 승인을 통해 정산이 개시됩니다.

자세한 정산시점 및 주기는 정산안내 탭을 참고하세요.

월간 정산금액 산정 기준
결제 수수료 마진
  • 당월 결제분에 대한 결제 수수료 마진에 대해 익월중 지급
유료 API 이용 마진
  • 선불 결제 시 : 당월분 이용 마진 금액이 익월중 지급
  • 후불 결제 시 : 당월분 이용금액이 익월말까지 납부됨에 따라, 후불 결제분에 대한 파트너 마진은 익월중 지급
정산완료 마진수익에 대한 정산금액이 지급되면 [정산완료]로 표시됩니다.
계산서 처리 및 원천징수 관련사항
  • 법인/개인사업자인 파트너의 경우 세금계산서 발행
  • 당월 1일~말일까지의 마진수익에 대해 익월 10일에 세금계산서 자동 발행 (발행기준일 : 익월 1일)
  • 익월중 일괄 정산입금
  • 개인 리셀러의 경우 부가세 및 원천징수세액 3.3% 공제 후 지급
