[가입] 누구나 가입할 수 있나요?
개발자, 에이전시, 프리랜서 등 개발 관련 업체 또는 종사자분들만 가입 가능합니다.
일반 영업자분들은 가입이 제한됩니다.
[정산] 정산일은 언제인가요?
결제 수수료 마진 : 당월 결제분에 대한 결제수수료 마진에 대해 익월말 지급
유료 API 이용 마진 : 당월분 이용 마진 금액이 익익월말에 지급
[마진] 마진설정은 어떻게 하나요?
관리자 접속 후 '설정 > 기본 마진 설정' 메뉴에서 신규판매점이 등록될 때 적용할 기본결제마진과 기본API마진을 설정할 수 있습니다.
[API 연동] 본인인증 에러 발생 시 창이 종료됩니다.
본인인증 연동 시 '인증에러/실패 시 리턴 URL'을 작성하지 않은 경우 에러 발생 시 창이 그대로 종료되는 현상이 발생합니다.
'본인인증 API > 연동설정' 메뉴에서 '인증에러/실패 시 리턴 URL'을 작성하시기 바랍니다.
