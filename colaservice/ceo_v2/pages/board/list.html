이미지 문자 발송 관련 유의사항

전단지 페이지에 등록되는 이미지는 고객님 폰의 데이터를 소모하지 않습니다.

그러나, 홍보문자 내용 밖에 이미지가 첨부돼 발송될 경우에는 이용 중인 통신사의 데이터 용량을 사용하게 됩니다. (통신사 정책 상 모바일 데이터를 끄고 와이파이 상태에서 발송이 돼도 통신사 데이터를 사용하는 것으로 돼있습니다.)

만약 통신요금의 기본 데이터가 모두 소진된 상태에서 이미지 문자를 발송하실 경우에는 모두 발송실패가 됩니다.

따라서, 한달에 2기가, 5기가 등 통신요금 데이터 용량이 크지 않으실 경우에는 홍보문자에 이미지를 한장만 첨부하시거나, 이미지 첨부 기능을 사용하지 않으시는 게 낫습니다.

데이터 무제한 요금제를 쓰시면 3장까지 이미지를 마음껏 첨부해 발송하셔도 됩니다.
